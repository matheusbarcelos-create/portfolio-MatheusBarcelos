# Portfólio Pessoal | Matheus Barcelos de Almeida

Este repositório contém o código-fonte do meu portfólio web pessoal. Ele foi desenvolvido para apresentar minha trajetória como estudante de Engenharia de Computação na Universidade de Uberaba, destacando minhas habilidades, projetos práticos e atuações extracurriculares.

🔗 https://matheusbarcelos-create.github.io/Portfolio-MATHEUSBARCELOS/

## 🚀 Visão Geral do Projeto

O site foi construído com foco em design responsivo, interatividade e clareza, sendo dividido em três páginas principais:

* **Página Inicial (`index.html`)**: Apresentação do meu perfil, resumo das minhas especialidades tecnológicas (Java, Python, JavaScript, C, SQL, HTML, CSS) e acesso aos meus certificados.
* **Projeto H²O (`h2o.html`)**: Detalhamento do sistema inteligente de IoT que desenvolvi e apresentei no XVIII ENTEC. O projeto utiliza um ESP32 (C++), sensor de fluxo YF-S201 e a plataforma ThingSpeak para monitoramento e detecção de vazamentos de água em tempo real.
* **Atlética Uniube (`atletica.html`)**: Página dedicada a compartilhar minha experiência e aprendizados em liderança e trabalho em equipe atuando como Diretor de Integração na Associação Atlética Acadêmica de Engenharias.

## 🛠️ Tecnologias e Recursos Utilizados

* **HTML5**: Estruturação semântica de todo o conteúdo.
* **CSS3**: Estilização customizada (`style.css`) utilizando variáveis do CSS (`:root`), Flexbox, transições suaves, animações de flutuação (`@keyframes`) e media queries para responsividade em telas menores.
* **JavaScript (Vanilla)**: 
  * `mobilemenu.js`: Lógica para abertura e fechamento do menu lateral em dispositivos móveis, com foco na atualização de atributos de acessibilidade (ARIA).
  * `rev.js`: Configuração de animações de entrada utilizando a biblioteca externa ScrollReveal.
* **Bootstrap Icons**: Utilizado via CDN para renderização de ícones vetorizados em toda a interface.
